# Conway's Game Of Life

```
$ python gol.py
usage: gol.py [-h] [-p P] [-ww WW] [-hh HH] [-x X] [-y Y] [-d D] [-g G] [--alive ALIVE] [--dead DEAD]

optional arguments:
  -h, --help     show this help message and exit
  -p P           Initial pattern name
  -ww WW         Width of the board
  -hh HH         Height of the board
  -x X           X axis offset
  -y Y           Y axis offset
  -d D           Delay between generations
  -g G           Generations to live
  --alive ALIVE  Living cell character
  --dead DEAD    Dead cell character
```
